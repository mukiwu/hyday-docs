> * Hyday 是一款以**本机优先** (Local-first) 为核心理念的笔记与日记桌面应用程序。只需下载与你作业系统对应的安装档即可开始使用 *

## 下载应用程序

请选择最新的稳定版本以获得最佳体验：

-  [ 下载最新稳定版 (GitHub Releases) ](https://github.com/mukiwu/hyday/releases/latest) 
-  [ 查看所有版本纪录 ](https://github.com/mukiwu/hyday/releases) 

> **ℹ️ Info**
>  Hyday 目前仅透过 GitHub Releases 提供下载，尚未于 Homebrew、Mac App Store 或 Microsoft Store 上架。 

## 平台支援

   平台 支援程度 说明     macOS 12+ 主要支援* 提供 Apple Silicon(`arm64`) 与 Intel(`x64`) 两种建置版本   Windows 10 / 11 支援 NSIS 安装程序，并包含已签署的自动更新套件   

 **主要支援**：表示为开发团队日常使用的平台，因此对问题修复的优先权最高 

## macOS 安装步骤

1. 下载对应你电脑晶片的 `.dmg` 文件，M 系列 (M1/M2/M3/M4...) 请选 arm64；Intel 请选 x64。
2. 打开 `.dmg` 映像档，将 Hyday 图示拖移至 Applications(应用程序) 文件夹。
3. 从 Launchpad 启动 Hyday。

## Windows 安装步骤

1. 下载 `.exe` 安装程序。
2. 执行安装精灵并依照指示完成步骤。
3. 从**开始**功能表启动 Hyday。
