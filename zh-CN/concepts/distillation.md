> *人格蒸馏是把任何人物的**心智模型、表达方式与决策习惯**提炼成可运行的人格框架，让 AI 能用他们的方式替你思考、回应、辩论*

## 何谓人格蒸馏？

多数 AI 工具只能让你切换语气或角色提示，但 Hyday 的人格蒸馏更丰富，我们透过 4 个阶段，把人物的**思考结构**抽取出来，组成一份可立即接到 Hyday Agent 对话与圆桌讨论的 persona 文件。

蒸馏完成后可应用于：

- 与**Hyday Agent 对话**时切换不同的人格视角，例如让 AI 用 Naval Ravikant、张忠谋或费曼的思考方式回应
- 在**圆桌讨论**中让多位人格同桌辩论，补足单一视角的盲点
- 遇到关键决策时，先与特定人物进行讨论，补足决策盲点

> **ℹ️ Info**
> 免费版可以使用 Hyday 官方提供的 10 种人格进行切换与对话，永久免费没有任何限制圆桌讨论与客制化人格蒸馏为 VVIP 专属功能，详见[方案差异对照表](https://hyday.tw/docs/reference/plans)

## 蒸馏的输入

启动蒸馏时只需提供少量信息，系统会自行调研：

- **人物名称**(必填)：例如*Richard Feynman*、*张忠谋*、*Naval Ravikant*
- **Wikipedia 链接**(选填)：可加快 Phase 1 的事实校正
- **代表链接**(选填，最多 3 个)：例如此人最具代表性的访谈、演讲或著作
- **分类**(选填)：可从「新创与产品」「学习与知识」「故事与创作」等 10 个分类中指定，不选则由 AI 自动推断

## 4 阶段蒸馏流程

整段流程约需**5 至 10 分钟**，可在背景执行；中途可随时中止

### Phase 1：六维调研

AI 会在公开网络上做六个面向的搜索：

- **著作**(writings)、**对话**(conversations)、**表达 DNA**(expression-dna)
- **他者观点**(external-views)、**关键决策**(decisions)、**时间线**(timeline)

若某一维度公开资料明显不足，AI 会诚实标记为**资料薄弱**，并在最后的**诚实边界**段落中注明

### Phase 2：综合提炼

把六维调研压缩成一份结构化的人格轮廓，包含：

- **心智模型**：跨领域证据支撑的核心思考框架
- **决策启发法**：实际情境中怎么判断与取舍
- **表达 DNA**：句式、用词、节奏、幽默感、确定度、引用习惯
- **价值观与反模式**：追求什么、拒绝什么、内在张力
- **知识谱系**：受谁影响、影响了谁、在思想史中的定位
- **标志性引言**与**一句话精髓**

### Phase 3：人格文件构建

把 Phase 2 的结构化轮廓组装成完整的 markdown 文件

### Phase 4：品质验证

用三道测试题确保蒸馏品质：

- **一致性检查**：基本事实一致性
- **边界测试**：边界情境下是否仍维持人物该有的判断
- **表达品质**：表达口气是否符合 Phase 2 抽出的表达 DNA

## 为什么需要网络搜索？

Phase 1 必须做六个面向的即时调研，因此**强烈建议使用 CLI 后端**(Claude Code、Gemini CLI、Codex CLI)，这些后端内建搜索工具，可直接运作。若选用 API 后端，需额外配置搜索工具，否则 Phase 1 可能无法完成。

如何选择适合你的 CLI，请参考：**设置 → AI → Functions → 人格蒸馏**

## 蒸馏后可以做什么

- **启用 / 停用**：在「设置 → 人格蒸馏与管理」中管理所有人格
- **编辑**：可手动微调系统提示词或更新人格头像
- **切换对话视角**：与 Hyday Agent 对话时，可随时切换人格
- **圆桌讨论**：邀请多位蒸馏出的人格同桌辩论 (VVIP 功能)

## 隐私与运算成本

人格蒸馏呼叫的是**使用者自己选定的 AI Provider**，并只送出公开资料 (人物名称、提供的链接) 进行调研，不会把你的笔记内容上传到任何云端。如对隐私有极高要求，可选用本机端的 Ollama，但本机模型在六维搜索与综合推理上的表现会明显低于云端模型，可能导致 Phase 1 直接失败或 Phase 2 结构化品质下降。
