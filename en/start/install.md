> * Hyday is a desktop notes and journal app built around a **local-first** philosophy. Just download the installer for your operating system to get started. *

## Download the App

Choose the latest stable release for the best experience:

-  [ Download the Latest Stable Release (GitHub Releases) ](https://github.com/mukiwu/hyday/releases/latest) 
-  [ View All Release Notes ](https://github.com/mukiwu/hyday/releases) 

> **ℹ️ Info**
>  Hyday is currently only available through GitHub Releases. It is not yet listed on Homebrew, the Mac App Store, or the Microsoft Store. 

## Platform Support

   Platform Support Level Notes     macOS 12+ Primary support* Available in both Apple Silicon (`arm64`) and Intel (`x64`) builds   Windows 10 / 11 Supported NSIS installer with a signed auto-update package included   

 **Primary support**: the platform the development team uses daily, so it gets the highest priority for bug fixes 

## macOS Installation Steps

1. Download the `.dmg` file that matches your chip. Choose arm64 for M-series Macs (M1/M2/M3/M4...) and x64 for Intel.
2. Open the `.dmg` image and drag the Hyday icon into the Applications folder.
3. Launch Hyday from Launchpad.

## Windows Installation Steps

1. Download the `.exe` installer.
2. Run the installation wizard and follow the prompts.
3. Launch Hyday from the **Start** menu.
