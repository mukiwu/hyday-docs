> ***Try the sample folder**for a concise tour of Hyday's core features: journaling, LifeLog time marks, backlinks, the tag system, and entity references.*

## What's in the Sample Folder?

- **A journal entry**: shows how to log a full day using time marks like`>(09:00)`and`<(10:00)`
- **Interlinked notes**: demonstrates backlinks created with`[[note name]]`, letting information flow naturally
- **A set of tags**: shows how to organize topics with tags like`#work`or`#ideas`
- **Entity tagging**: uses`@Alice`or`@2025-plan`to connect related people, projects, and topics across notes

![The main view after opening the sample folder](https://hyday.tw/docs/screenshots/start-getting-started-folder-1.png)

## Suggested Walkthrough

1. **Start with today's journal entry**: see how LifeLog time marks turn scattered notes into a structured timeline
2. **Click a backlink**: tap any`[[note name]]`to see how Hyday jumps seamlessly between notes
3. **Check backlinks**: open a note and look at the panel at the bottom to see which other notes reference it
4. **Explore tags**: click any`#tag`to open the tag side panel and review notes on the same topic
5. **Follow an entity**: click any`@entity`to see all related notes and the lineage connecting them

## After the Walkthrough

- Try writing your own journal entry or note
- Delete the sample folder and create or open the folder you'll actually use

You can delete the sample folder anytime — it won't affect your real notes. Next, learn how to:[Open or Create a Folder](https://hyday.tw/docs/start/open-or-create-folder)
