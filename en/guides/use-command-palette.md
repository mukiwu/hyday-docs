> *Hyday's command palette lets you jump to any page or run any feature without leaving the keyboard.*

## 1. Key Shortcuts

PurposemacOSWindows**Command Palette**(run actions)`Cmd`+`Shift`+`P``Ctrl`+`Shift`+`P`**Global Search**(find notes and content)`Cmd`+`K``Ctrl`+`K`

Think of the**Command Palette**as the window for running**actions**, and**Global Search**as the entry point for finding**content**. They complement each other with a clear division of labor.

![Command palette UI](https://hyday.tw/docs/screenshots/guides-use-command-palette-1.png)

## 2. What the Command Palette Covers

- **Create content**: such as**Create new note**or**New journal entry**
- **Quick navigation**: chat with Hyday Agent or jump to journal, tasks, whiteboard, settings, or trash

## 3. What Global Search Can Find

- Note titles and full-text content
- Tag names
- Entities
- Semantic search powered by embeddings (beyond keyword matches)

## 4. In-Editor Triggers

Beyond the global shortcuts above, the editor supports several character-based triggers:

- `/`: open the slash command menu.
- `[``[`: insert a backlink.
- `#`: tag.
- `@`: mention an entity.

## 5. The Starting Point for Everything

When you forget where a feature lives, press`Cmd (Ctrl)`+`Shift`+`P`and start typing — the palette usually surfaces the right action immediately.
