> *人格蒸餾是把任何人物的**心智模型、表達方式與決策習慣**提煉成可運行的人格框架，讓 AI 能用他們的方式替你思考、回應、辯論*

## 何謂人格蒸餾？

多數 AI 工具只能讓你切換語氣或角色提示，但 Hyday 的人格蒸餾更豐富，我們透過 4 個階段，把人物的**思考結構**抽取出來，組成一份可立即接到 Hyday Agent 對話與圓桌討論的 persona 檔案。

蒸餾完成後可應用於：

- 與**Hyday Agent 對話**時切換不同的人格視角，例如讓 AI 用 Naval Ravikant、張忠謀或費曼的思考方式回應
- 在**圓桌討論**中讓多位人格同桌辯論，補足單一視角的盲點
- 遇到關鍵決策時，先與特定人物進行討論，補足決策盲點

> **ℹ️ Info**
> 免費版可以使用 Hyday 官方提供的 10 種人格進行切換與對話，永久免費沒有任何限制圓桌討論與客製化人格蒸餾為 VVIP 專屬功能，詳見[方案差異對照表](https://hyday.tw/docs/reference/plans)

## 蒸餾的輸入

啟動蒸餾時只需提供少量資訊，系統會自行調研：

- **人物名稱**(必填)：例如*Richard Feynman*、*張忠謀*、*Naval Ravikant*
- **Wikipedia 連結**(選填)：可加快 Phase 1 的事實校正
- **代表連結**(選填，最多 3 個)：例如此人最具代表性的訪談、演講或著作
- **分類**(選填)：可從「新創與產品」「學習與知識」「故事與創作」等 10 個分類中指定，不選則由 AI 自動推斷

## 4 階段蒸餾流程

整段流程約需**5 至 10 分鐘**，可在背景執行；中途可隨時中止

### Phase 1：六維調研

AI 會在公開網路上做六個面向的搜尋：

- **著作**(writings)、**對話**(conversations)、**表達 DNA**(expression-dna)
- **他者觀點**(external-views)、**關鍵決策**(decisions)、**時間線**(timeline)

若某一維度公開資料明顯不足，AI 會誠實標記為**資料薄弱**，並在最後的**誠實邊界**段落中註明

### Phase 2：綜合提煉

把六維調研壓縮成一份結構化的人格輪廓，包含：

- **心智模型**：跨領域證據支撐的核心思考框架
- **決策啟發法**：實際情境中怎麼判斷與取捨
- **表達 DNA**：句式、用詞、節奏、幽默感、確定度、引用習慣
- **價值觀與反模式**：追求什麼、拒絕什麼、內在張力
- **知識譜系**：受誰影響、影響了誰、在思想史中的定位
- **標誌性引言**與**一句話精髓**

### Phase 3：人格檔案構建

把 Phase 2 的結構化輪廓組裝成完整的 markdown 檔案

### Phase 4：品質驗證

用三道測試題確保蒸餾品質：

- **一致性檢查**：基本事實一致性
- **邊界測試**：邊界情境下是否仍維持人物該有的判斷
- **表達品質**：表達口氣是否符合 Phase 2 抽出的表達 DNA

## 為什麼需要網路搜尋？

Phase 1 必須做六個面向的即時調研，因此**強烈建議使用 CLI 後端**(Claude Code、Gemini CLI、Codex CLI)，這些後端內建搜尋工具，可直接運作。若選用 API 後端，需額外配置搜尋工具，否則 Phase 1 可能無法完成。

如何選擇適合你的 CLI，請參考：**設定 → AI → Functions → 人格蒸餾**

## 蒸餾後可以做什麼

- **啟用 / 停用**：在「設定 → 人格蒸餾與管理」中管理所有人格
- **編輯**：可手動微調系統提示詞或更新人格頭像
- **切換對話視角**：與 Hyday Agent 對話時，可隨時切換人格
- **圓桌討論**：邀請多位蒸餾出的人格同桌辯論 (VVIP 功能)

## 隱私與運算成本

人格蒸餾呼叫的是**使用者自己選定的 AI Provider**，並只送出公開資料 (人物名稱、提供的連結) 進行調研，不會把你的筆記內容上傳到任何雲端。如對隱私有極高要求，可選用本機端的 Ollama，但本機模型在六維搜尋與綜合推理上的表現會明顯低於雲端模型，可能導致 Phase 1 直接失敗或 Phase 2 結構化品質下降。
