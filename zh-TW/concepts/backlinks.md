> * 雙向連結 (Backlinks) 能讓筆記之間建立相互引用的關聯。只需在其中一篇筆記中建立連結，兩者即可在系統中自動串聯。 *

## 如何建立連結？

在編輯器中輸入 `[['`，系統會自動跳出筆記選擇清單。可以直接選擇既有筆記，或輸入文字進行搜尋：

```
`這個想法與 [[2025-04 春季計畫]] 密切相關`
```

## **雙向**連結的價值

當在**筆記 A**中寫入 `[[筆記 B]]`，在開啟**筆記 B**時，右側面板會自動顯示**筆記 A 引用了此篇內容**。**無須手動在兩篇筆記間重複操作**，關聯性會由系統自動建立與維護。

![筆記頁面右側的反向連結面板](https://hyday.tw/docs/screenshots/concepts-backlinks-1.png)

## 連結至日記條目

除了普通筆記外，也可以直接連結至特定日期的日記：

```
`這項決定隨後在 [[2026-05-09]] 的紀錄中進行了修正`
```

> **⚠️ Warning**
>  Hyday 目前支援以**整篇筆記**或**單日日記**為單位的連結。暫不支援 Obsidian 風格的區塊級 (Block-level) 錨點連結 (如 `[[2026-05-09#10:30]]`)。 

## 若修改檔名，連結會失效嗎？

若手動更改了筆記的實體檔名，原本指向該檔案的 `[[]]` 語法會自動連動更新連結

## 尋找**孤島筆記**

透過設定 → 管理庫，可以看到沒有被其他筆記引用的檔案，對於定期整理知識庫、重建資料關聯非常有幫助
