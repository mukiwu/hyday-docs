> * Hyday 是一款以**本機優先** (Local-first) 為核心理念的筆記與日記桌面應用程式。只需下載與你作業系統對應的安裝檔即可開始使用 *

## 下載應用程式

請選擇最新的穩定版本以獲得最佳體驗：

-  [ 下載最新穩定版 (GitHub Releases) ](https://github.com/mukiwu/hyday/releases/latest) 
-  [ 查看所有版本紀錄 ](https://github.com/mukiwu/hyday/releases) 

> **ℹ️ Info**
>  Hyday 目前僅透過 GitHub Releases 提供下載，尚未於 Homebrew、Mac App Store 或 Microsoft Store 上架。 

## 平台支援

   平台 支援程度 說明     macOS 12+ 主要支援* 提供 Apple Silicon(`arm64`) 與 Intel(`x64`) 兩種建置版本   Windows 10 / 11 支援 NSIS 安裝程式，並包含已簽署的自動更新套件   

 **主要支援**：表示為開發團隊日常使用的平台，因此對問題修復的優先權最高 

## macOS 安裝步驟

1. 下載對應你電腦晶片的 `.dmg` 檔案，M 系列 (M1/M2/M3/M4...) 請選 arm64；Intel 請選 x64。
2. 開啟 `.dmg` 映像檔，將 Hyday 圖示拖移至 Applications(應用程式) 資料夾。
3. 從 Launchpad 啟動 Hyday。

## Windows 安裝步驟

1. 下載 `.exe` 安裝程式。
2. 執行安裝精靈並依照指示完成步驟。
3. 從**開始**功能表啟動 Hyday。
