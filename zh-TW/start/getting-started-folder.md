> * **體驗範例資料夾**透過精簡的內容，帶領快速熟悉 Hyday 的核心功能：日記紀錄、LifeLog 時間標記、雙向連結、標籤系統與人事物管理 *

## 範例資料夾包含什麼？

- **一篇日記檔案**：示範如何使用 `>(09:00)` 與 `<(10:00)` 等時間標記來記錄全日行程
- **相互關聯的筆記**：展示透過 `[[筆記名稱]]` 建立的雙向連結，讓資訊自然流動
- **標籤組合**：示範利用 `#工作` 或 `#想法` 等標籤進行主題分類
- **實體標記**：透過 `@Alice` 或 `@2025計畫` 跨筆記串聯相關的人、事、物

![範例資料夾開啟後的主畫面視圖](https://hyday.tw/docs/screenshots/start-getting-started-folder-1.png)

## 建議探索步驟

1. **從今天日記開始**：體驗 LifeLog 時間標記如何將瑣碎的紀錄轉化為結構化的時間線
2. **點擊雙向連結**：點選任何 `[[筆記名稱]]`，查看 Hyday 如何在不同筆記間無縫跳轉
3. **檢視反向連結**：點開筆記後，查看底部面板，了解有哪些其他筆記正引用當前內容
4. **探索標籤**：點擊任何 `#標籤`，開啟標籤右側面板以檢視同類主題
5. **追蹤人事物**：點擊任何 `@人事物`，查看所串接的所有相關筆記與脈絡

## 完成初步體驗後

- 試著撰寫一則屬於自己的日記或筆記
- 刪除範例資料夾，並建立或開啟正式使用的筆記資料夾

 可隨時刪除範例資料夾，這不會影響正式筆記內容。接下來可以了解如何：[開啟或建立資料夾](https://hyday.tw/docs/start/open-or-create-folder)
