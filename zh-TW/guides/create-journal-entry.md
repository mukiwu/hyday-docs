> * 日記記錄生活的點滴，每天系統會自動新增一篇以日期命名的日記 *

## 1. 開啟今日日記

在側邊欄點擊**日記**，若今日尚未建立日記，Hyday 會自動建立一個 `YYYY-MM-DD.md` 檔案 (例如 `2026-05-09.md`)

## 2. 隨手紀錄，無須拘束

日記內容不一定要有嚴謹的結構。可以輕鬆記錄任何瑣碎想法：

```
``
```

## 3. 進階：運用**區塊**組織內容

在撰寫時，每換一個新段落，Hyday 都會將其視為一個獨立的**區塊 (Block)**。可隨時拖曳這些區塊來調整紀錄的順序

詳細內容請參考：[使用 LifeLog 追蹤一整天](https://hyday.tw/docs/guides/track-a-day-with-lifelog)

## 5. 翻閱與檢索歷史日記

可透過側邊欄日記分頁中的「活動紀錄」點擊任一日期，也可以使用全域搜尋 (`⌘`+`K`) 輸入日期，快速跳轉至特定天的紀錄

> **ℹ️ Info**
>  簡短的三言兩語或詳盡的數千長文都是日記的一種，重點在於養成**在當天留下紀錄**的習慣，這些零碎資訊將在未來展現出驚人的關聯價值
